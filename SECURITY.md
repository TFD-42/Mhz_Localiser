# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |
| older releases | ❌ |

## Ethical Use

This tool is intended for:
- Legitimate RF survey and site analysis
- Educational RF experimentation
- Amateur radio fox-hunting
- Finding your own lost devices

**Do not use this tool to track people, intercept communications, or operate on frequencies you are not licensed to use.** RF regulations vary by country — the user is solely responsible for compliance with local laws.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not open a public issue**.

Report it privately via GitHub's Security Advisory:
👉 https://github.com/TFD-42/Geo-Flip/security/advisories/new

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You will receive a response within **72 hours**. We appreciate responsible disclosure.
